# PART 1: ONBOARDING FLOW

## Overview

The onboarding must accomplish:
1. Capture training architecture preferences
2. Understand goals and timeline
3. Assess current fitness level
4. Configure equipment access
5. Set up Apple Watch
6. Generate first workout with AI explanation
7. Build excitement and trust

**Target Duration:** 3-5 minutes (can skip and refine later)
