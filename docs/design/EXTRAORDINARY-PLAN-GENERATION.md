## PART 5: TRAINING METHODOLOGY LIBRARY

### The AI Can Choose From MULTIPLE Proven Approaches:

**Methodology 1: POLARIZED TRAINING (Research-Backed)**
```
Philosophy: 80% easy (Zone 1-2), 20% hard (Zone 4-5), avoid "grey zone" (Zone 3)

Best for:
- Aerobic development priority
- Sustainable long-term gains
- Injury-prone athletes
- Older athletes (35+)

Week Structure (4 days):
- Day 1: Easy run 45min Z2
- Day 2: Strength-endurance circuit 60min
- Day 3: Rest or active recovery
- Day 4: Intervals 40min (Z4-Z5)
- Day 5: Stations + short run
- Day 6: Long run 70min Z2
- Day 7: Rest

Why it works: Maximum stress with maximum recovery
```

**Methodology 2: CONJUGATE METHOD (Westside Barbell Adapted)**
```
Philosophy: Rotate exercises frequently, max effort + dynamic effort days

Best for:
- Strength-focused athletes
- Advanced lifters
- Stale progress (need variety)
- Power development

Week Structure (4 days):
- Day 1: Max Effort Lower (work up to heavy 3-5 rep squat variation)
- Day 2: Dynamic Effort Upper (speed bench, explosive push work)
- Day 3: Easy run or sled work
- Day 4: Max Effort Upper (heavy pressing or pulling)
- Day 5: Dynamic Effort Lower (box jumps, speed squats)
- Day 6: Hybrid conditioning (stations + running)
- Day 7: Rest

Why it works: Avoids accommodation, builds multiple strength qualities
```

**Methodology 3: NORWEGIAN METHOD (Ingebrigtsen Brothers)**
```
Philosophy: High volume of lactate threshold work, double threshold days

Best for:
- Running-focused athletes
- High work capacity
- Race pace specificity
- Aggressive timeline

Week Structure (6 days, often 2 sessions/day):
- Mon AM: Easy run 40min | PM: Threshold 5x1000m
- Tue AM: Tempo run 30min | PM: Strength circuit
- Wed AM: Easy run 30min | PM: Easy run 30min
- Thu AM: Easy run 40min | PM: Threshold 4x1500m
- Fri: Rest or easy 30min
- Sat: Long run 90min
- Sun: Stations + short run

Why it works: Massive threshold volume improves race pace
```

**Methodology 4: BLOCK PERIODIZATION (Issurin Model)**
```
Philosophy: Concentrate on 1-2 abilities per block, sequence for peak

Best for:
- Specific race date
- Systematic improvement
- Science-based athletes
- Long-term planning (16+ weeks)

Block 1 - Accumulation (4-6 weeks):
- High volume, low intensity
- GPP, movement quality
- Build work capacity

Block 2 - Intensification (3-4 weeks):
- Reduce volume, increase intensity
- HYROX-specific work
- Stations + compromised running

Block 3 - Realization (2-3 weeks):
- Low volume, race-specific
- Full simulations
- Peak power/speed

Block 4 - Taper (1-2 weeks):
- Minimal volume, sharp work
- Race day preparation
- Arrive fresh and fast

Why it works: Systematic targeting of adaptations
```

**Methodology 5: DAILY UNDULATING PERIODIZATION (DUP)**
```
Philosophy: Vary intensity daily within same week

Best for:
- Busy schedules
- Need for variety
- General fitness athletes
- No specific race date

Week Structure:
- Mon: Heavy strength (3-5 reps, 85%+ 1RM)
- Tue: Interval running (high intensity, low volume)
- Wed: Strength-endurance (12-20 reps, 60% 1RM)
- Thu: Tempo run (moderate intensity)
- Fri: Power work (3-5 reps, 70% but FAST)
- Sat: Long hybrid (stations + long run)
- Sun: Rest

Why it works: Constant variation prevents staleness
```

**Methodology 6: AUTO-REGULATED PROGRESSION (RIR/RPE Based)**
```
Philosophy: Training load determined by daily readiness, not predetermined

Best for:
- Variable schedule
- High stress life
- HRV-responsive athletes
- Adaptive AI perfect match

How it works:
- Every morning: Readiness check (HRV, sleep, subjective)
- High readiness (85+) → Hard session, push limits
- Moderate (65-84) → Standard prescribed work
- Low (<65) → Easy day or rest, regardless of plan

Week Structure:
- Planned as normal
- But adjusted daily based on readiness
- AI learns optimal load over time

Why it works: Matches training to recovery capacity
```
