# PART 1: UNDERSTANDING HYROX

## 1.1 What is HYROX?

HYROX is a standardized indoor fitness race combining **8 x 1km runs** with **8 functional workout stations**. The format is identical worldwide, enabling global rankings.

### Race Format

```
START → 1km Run → SkiErg (1000m) → 1km Run → Sled Push (50m) →
1km Run → Sled Pull (50m) → 1km Run → Burpee Broad Jumps (80m) →
1km Run → Rowing (1000m) → 1km Run → Farmers Carry (200m) →
1km Run → Sandbag Lunges (100m) → 1km Run → Wall Balls (100 reps) → FINISH
```

### Divisions & Weight Standards

| Division | Gender | Sled Push | Sled Pull | Farmers | Lunges | Wall Ball |
|----------|--------|-----------|-----------|---------|--------|-----------|
| Open Women | F | 102kg | 78kg | 2x16kg | 10kg | 4kg |
| Open Men | M | 152kg | 103kg | 2x24kg | 20kg | 6kg |
| Pro Women | F | 152kg | 103kg | 2x24kg | 20kg | 6kg |
| Pro Men | M | 202kg | 153kg | 2x32kg | 30kg | 9kg |

**Additional Divisions:** Doubles (pairs), Relay (4-person teams)

### Typical Completion Times

| Level | Men | Women |
|-------|-----|-------|
| Elite | Sub-60 min | Sub-65 min |
| Competitive | 65-75 min | 70-80 min |
| Intermediate | 75-90 min | 80-95 min |
| Recreational | 90-120 min | 95-130 min |

## 1.2 Physiological Demands

Understanding the science is critical for effective AI programming.

### Energy Systems Required
- **60-70% Aerobic** - Sustained effort over 60-90+ minutes
- **20-30% Lactate Threshold** - Station work drives lactate to 8-12 mmol/L
- **10-15% Anaerobic** - Short bursts during station transitions

### Key Physical Qualities
1. **Cardiovascular Endurance** - VO2max is the strongest predictor of performance
2. **Strength-Endurance** - Maintain power output over high rep ranges
3. **Grip Strength** - Critical for farmers carry, sled pull, rowing
4. **Core Stability** - All 8 stations demand trunk control
5. **Mental Resilience** - 60-90+ minutes of sustained discomfort

### Common Athlete Weaknesses
| Weakness | % of Athletes | Training Solution |
|----------|---------------|-------------------|
| Underestimating aerobic base | 80% | Zone 2 volume emphasis |
| Poor pacing (too fast start) | 75% | Race simulation practice |
| Training in isolation | 70% | "Compromised running" sessions |
| Grip strength failures | 60% | Grip-specific work |
| Burpee/wall ball fatigue | 50% | High-rep conditioning |
